```java
public static Figure of(List<Point> points) {
    	if(points.size() == 2){
        	new Line(points);
        }else if(points.size() == 3){
            new Triangle(points);
        }else if(points.size() == 4){
            new Rectangle(points);
        }
    }
```

```java
public static Figure of(List<Point> points) {
        Figure figure = points.get(0);
        for (int i = 1; i < points.size(); i++) {
            figure = figure.addPoint(points.get(i));
        }
        return figure;
    }
```
