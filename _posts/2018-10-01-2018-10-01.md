### 묻지말고 시켜라

```java
@Test
public void add(){
    Number number = new Number(5);
    number.add(2);
    number.add(3);
    number.add(5);
}

@Test
public void add1(){
    number.get() + 2;	//로직의 중복
    number.get() + 3;
    number.get() + 5;
}
```
